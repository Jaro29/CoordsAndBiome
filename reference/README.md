# Hytale API Reference

Źródło z `/home/jaro/Dokumenty/hytale-server/` – API potrzebne do modu CoordsAndBiome (HUD z koordynatami i biomem).

Skopiowane pakiety:
- `component/` – DelayedEntitySystem, EntityTickingSystem, Query, Archetype
- `event/` – EventRegistry
- `logger/` – HytaleLogger
- `server/core/` – plugin, command, event, entity, ui, Message
- `server/worldgen/` – BiomeDataSystem, ChunkGenerator, ZoneBiomeResult, Biome

Przykład UI:
- `coords_hud_example.ui` – szablon HUD z Label #CoordsLabel, Anchor (Bottom: 100)
